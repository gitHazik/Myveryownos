# MYveryOWNcustomOS

This is a web-based operating system I built from scratch using nothing but vanilla JavaScript, HTML, and CSS. 

My goal was to create a clean, functional workspace that stays out of your way. I’ve leaned into a glassmorphic aesthetic to keep the UI feeling modern and light, drawing quite a bit of inspiration from the look and feel of various Linux desktop environments.

---

### Features

* **Pure Tech**: Built entirely with vanilla JS/HTML/CSS—no heavy frameworks or bloat.
* **Minimalist Interface**: Designed to keep your workspace uncluttered and focused.
* **Glassmorphic Design**: Uses semi-transparent, blurred layers to create depth without the visual noise.
