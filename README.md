# myVeryOwnOS

A browser based desktop OS built for a hackathon. No frameworks, no libraries just HTML, CSS, and vanilla JS.

---

## Tech

- HTML5
- CSS3 (glassmorphism, custom properties, backdropfilter)
- Vanilla JavaScript 

---

## What it does

- **Boot screen** — linux-style boot log with a progress bar before the desktop loads
- **Windowed apps** — draggable, resizable, minimise/maximise/close, with proper z-index focus management
- **Notepad** — type, change font/size, save as .txt, open local files
- **File Explorer** — virtual filesystem with sidebar navigation and folder/file creation
- **Browser** — working iframe browser with tabs and a URL bar (search fallback included)
- **Settings** — change accent colour, blur intensity, glass opacity, wallpaper, 12/24h clock
- **Taskbar** — pinned apps, system tray, live clock
- **Right-click context menu** on the desktop
